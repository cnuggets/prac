# PracJS
A Simple but Powerful Web UI Framework & Toolkit

## Quick Start
- Create project

```
mkdir YOUR-PROJECT
cd YOUR-PROJECT
npm init

```
- Install prac

```
npm install pracjs --save-dev
sudo npm install pracjs -g
```
- Intialize prac project

```
# responsive web
prac init
# or mobile
prac init -m
```

- Start a web server

```
prac run -p 8080
```
